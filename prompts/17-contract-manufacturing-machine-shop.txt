BUSINESS-SPECIFIC OPS — CONTRACT MANUFACTURER / MACHINE SHOP (B2B, for OEMs)
================================================================
WHAT "AFTER THE SITE" MEANS
The site presents capabilities (tolerance / material / lead-time facts) and
captures RFQs that involve an NDA and a drawing/part file. Buyers are OEM
engineers/purchasers; the relationship is long, technical, and IP-sensitive. The
ops job is to acknowledge the RFQ and handle the NDA + drawing securely, quote,
win the PO, deliver to spec, then earn a reference (B2B reviews are usually
references/case studies, not Google stars).

LEAD INTENTS (intents[])
- "rfq" (P1) — request for quote with a part/drawing.
- "nda" (P1) — needs an NDA before sharing files.
- "capability" (P2) — can-you-make-this question.
- "general" (P3).

CRM PIPELINE (stages[])
rfq -> nda -> quote -> po -> production -> shipped -> won / lost.

FOLLOW-UP (followUp)
Fire immediately. Acknowledge the RFQ and give SECURE drawing-submission
instructions — never invite proprietary files or specs into an open email
thread; link to a secure upload and offer the NDA first. Confirm material /
tolerance / quantity / target lead-time. delayHours: 0.

REVIEW REQUEST (reviewRequest)
trigger: "job-complete" mapped to SHIPPED + accepted. Ask for a REFERENCE /
case-study permission or a Google/industry review as appropriate — many OEM
buyers can't post public reviews, so default to a private reference request and
make the public review optional. Platform: GOOGLE_REVIEW_URL (optional). Never
fabricate a testimonial.

INVOICING / REMINDERS (invoice)
PO-based invoicing on NET terms (net-30 / net-60). reminderDays measured from the
invoice/PO terms; keep the tone B2B-professional. Repurpose for a reorder /
blanket-PO nudge on recurring parts.

COMPLIANCE FOR OUTBOUND
Truthful certification claims only — ISO 9001 / IATF 16949 / AS9100 as TODO
tokens until verified; never invent. OHSA / WSIB. Controlled-goods / export
(ITAR-equivalent) handling noted only if applicable. NDA + IP confidentiality
respected. CASL consent + unsubscribe.

VERTICAL AUTOMATIONS (optional n8n additions)
- Gate the public review step behind manual approval; default the "review" to a
  private reference request.
- An RFQ-aging nudge (follow up on open quotes after N days) reusing the
  schedule trigger.
